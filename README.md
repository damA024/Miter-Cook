# MisterCook

### Installer

#### Pour utiliser le projet localement, suivez ces étapes :

1. Clonez ce dépôt sur votre machine en utilisant la commande suivante :

```bash
git clone https://github.com/damA024/Miter-Cook.git
```

1. Naviguez dans le dossier du projet :

```bash
cd nom_du_projet
```

1. Installez toutes les dépendances du projet :

```bash
npm install
```

### Contribuer

1. Pour mettre à jour votre copie locale avec les dernières modifications du dépôt distant.

```bash
git pull
```

2.Pour préparer des modifications à être incluses dans le prochain commit.

```bash
git add .
```

3.Pour enregistrer les modifications dans l'historique du dépôt.

```bash
git commit -m "Message de commit descriptif"
```

4. Pour envoyer les modifications locales vers le dépôt distant.

```bash
git push
```
